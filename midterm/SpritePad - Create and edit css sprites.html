About SpritePad

SpritePad is your free and easy-to-use web app for creating CSS spritemaps. It only needs seconds:
Drag & drop your images onto the grid and have it immediately available as one PNG + CSS code.
No fiddling in Photoshop, no manual assignment of CSS styles, no long-winded setups,
SpritePad makes creating spritemaps as easy as writing your next e-mail.
